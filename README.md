# Sales Management System

## Project Overview

A Sales Management Dashboard built using Python, PostgreSQL, and Streamlit to manage sales transactions, payment tracking, branch performance, and business analytics.

---

## Features

- Sales Dashboard
- KPI Monitoring
- Branch Performance Analysis
- Payment Tracking
- Financial Calculations
- PostgreSQL Database Integration
- Automated Database Triggers
- Interactive Streamlit Interface

---

## Tech Stack

- Python
- PostgreSQL
- Streamlit
- Pandas
- Matplotlib

---

## Database Tables

- branches
- customer_sales
- payments
- payment_splits

---

## Key Metrics

- Total Sales
- Total Received Amount
- Outstanding Balance
- Branch-wise Performance
- Sales Trends

## Installation

1. Clone the repository

git clone <repository-link>

2. Install dependencies

pip install -r requirements.txt

3. Run Streamlit

streamlit run app.py
